Carte¶

Module pour la génération d’une carte interactive entre deux stations.

Ce script permet à l’utilisateur de visualiser le chemin le plus court entre deux stations de vélos à Montpellier. Il utilise des données CSV contenant les trajets de vélos partagés, les coordonnées des stations stockées dans un fichier JSON, et un graphe routier OSMNX pour calculer le chemin optimal.

Dépendances : - osmnx - folium - networkx - json - pandas - geopy - time - sys - os
